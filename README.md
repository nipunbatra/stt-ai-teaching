## Building Slides

### Prerequisites

Install Quarto: https://quarto.org/docs/get-started/

### Using Makefile (Recommended)

```bash
# Build everything (HTML + PDF slides + PDF handouts)
make all

# Build specific formats
make html                     # HTML slides only
make pdf                      # PDF slides only (decktape)
make handout                  # PDF handouts only (beamer)

# Build specific slide in all formats
make data-collection-labeling

# Or use Quarto directly for site
quarto render

# List available slides
make list

# Clean generated files
make clean
```

### Manual Build

**1. HTML Slides (RevealJS):**
```bash
quarto render data-collection-labeling.qmd --to revealjs
```

**2. PDF Slides (from RevealJS HTML using decktape):**
```bash
# First build HTML
quarto render data-collection-labeling.qmd --to revealjs
# Then convert to PDF presentation format
npx decktape reveal data-collection-labeling.html data-collection-labeling.pdf --chrome-arg=--no-sandbox
```

**3. PDF Handout (Beamer document format):**
```bash
quarto render data-collection-labeling.qmd --to beamer --output data-collection-labeling-handout.pdf
```

**Navigation (HTML slides):**
- Arrow keys / Space: Next slide
- `F`: Fullscreen
- `S`: Speaker notes
- `C`: Chalkboard (draw on slides)

### GitHub Actions (Automatic)

GitHub Actions automatically builds and deploys on every push.

**Workflow:**
1. Edit `.qmd` files
2. Build locally: `make all` (optional, for local preview)
3. Commit and push: `git add *.qmd *.pdf *-handout.pdf && git commit && git push`
4. GitHub Actions: Builds HTML and deploys to Pages automatically

**What's committed:** Source files (`.qmd`), PDF slides, and PDF handouts
**What's generated by GitHub Actions:** HTML files and index page

## Editing Slides

Edit the `.qmd` file directly - it's Markdown with special syntax for slides.

**Slide breaks:**
- `#` - New section (title slide)
- `##` - New slide

**Incremental lists:**
```markdown
::: incremental
- Item 1
- Item 2
:::
```

**Code blocks:**
````markdown
```python
# Your code here
```
````

**Columns:**
```markdown
::: columns
::: {.column width="50%"}
Left content
:::
::: {.column width="50%"}
Right content
:::
:::
```
