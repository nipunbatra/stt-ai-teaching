# Lab: Interactive AI Demos

**Objective**: Build and deploy an interactive web app for an AI model.

## Task 1: Streamlit Basics
- Install Streamlit.
- Create a "Hello World" app.
- Add widgets (slider, text input) to control parameters.

## Task 2: Model Integration
- Load a pre-trained model (e.g., Hugging Face pipeline for sentiment analysis).
- Connect widgets to model inputs.
- Display model output.

## Task 3: Deploy to Cloud
- Push code to GitHub.
- Connect to Streamlit Community Cloud.
- Verify public URL works.
